About the Developers - Jitin, Vishal & Roshan

Our Medicine Recommendation System is a collaborative effort of three enthusiastic developers...

Jitin – Machine Learning Engineer

Jitin played a pivotal role in building the core functionality of the system by developing and training the machine learning model. He used scikit-learn, a robust machine learning library, and Jupyter Notebook for data analysis, model evaluation, and optimization. His expertise in machine learning ensured that the system provides accurate and reliable medicine recommendations based on user input....

Vishal – Data Analyst & UI Designer

Vishal was responsible for the foundational elements of the project. He meticulously collected, cleaned, and prepared the dataset used to train the recommendation model. In addition, he worked on designing an intuitive and user-friendly interface, ensuring a seamless user experience. His attention to detail in managing assets and creating an engaging UI greatly enhanced the overall functionality and appearance of the system....

Roshan – Backend Developer

Roshan took charge of integrating the machine learning model with the web interface using Flask, a lightweight and powerful web framework. He successfully connected the Jupyter Notebook file with the website, ensuring real-time interaction between the user and the model. His backend development skills helped bridge the gap between the data science and user experience aspects of the project, making the system fully functional and accessible....
